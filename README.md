# I2C-tools（micro:bit 用 I2C 汎用拡張）

micro:bit で I2C センサーを扱うためのシンプルな拡張機能です。  
アドレスとレジスタを指定して、1バイト書き込み・1バイト読み取り・2バイト読み取りができます。

---

## 使い方（MakeCode）

MakeCode の「拡張機能」→「URL から追加」で、次の URL を入力してください。

https://github.com/keikyufun/I2c-tools#v0

読み込むと「I2Cツール」というカテゴリが追加されます。

---

## 追加されるブロック

- I2C 書き込み（アドレス・レジスタ・値）
- I2C 読み取り（1バイト）
- I2C 読み取り（2バイト）
